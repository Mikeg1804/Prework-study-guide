Git

  • git hub is a repository which help to maintain version control when coding
  • git bash is a Coment Line to command the computer
  • command: "git status" checks what branch we are currently on
  • command: "git checkout -b branch-name" creates a new branch and switches to it
  • command: "git code ." opens the index.html file for me to start coding
  • command: "git add -A to commit all files in the working branch.
  • command: "git commit -M" to associate a message with our commit
  • command: "cd (folder)" way to navigate to folders in git bash
  • switch to the main branch by entering the following Git command "git checkout main"
  • Git command to create a new branch named feature/add-html " git checkout -b feature/add-html"
  • command in your terminal to stage your changes "git add -A"
  • Use the following command in your terminal to commit the changes "git commit -m"
  • command in your terminal to pull in the base branch "git pull origin main"
  • push up all of our changes by using the following command: "git push origin feature/add-html"
  • See module 3.1.6 on how to push branch back to main on Github
